This plugin creates an online barcode reception system to your WordPress website.

== Description ==

Barcode Receptionは、WordPressでオンライン・バーコード受付システムを構築するプラグインです。
ログインユーザのバーコードを表示し、バーコードリーダーで受付を行うことができます。

= 特徴 =
* WordPressの登録ユーザーの受付を行うことができます
* ユーザのバーコード表示画面、バーコード受付画面、受付済みユーザ一覧を表示することができます
* 一般的な一次元バーコードリーダーに対応しています
* バーコードの値はユーザーID＋オフセット値（任意の値）です

詳細な使用方法はこちらです: http://residentbird.main.jp/bizplugin/plugins/barcode-reception/

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
